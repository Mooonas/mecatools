# WebSite-Mecathronics
Personal showcase and resource hub for engineering students. This site provides a collection of field-tested tools and hardware for Mechatronics projects, featuring affiliate links for recommended gear and equipment.

#MechaTools
##Propósito del Proyecto
MechaTools es una plataforma diseñada por y para estudiantes de Mecatrónica. El objetivo es ofrecer un catálogo curado de herramientas de precisión —desde multímetros hasta kits de robótica— que han sido seleccionadas bajo un criterio técnico riguroso. A diferencia de las recomendaciones genéricas, todos los productos listados han sido evaluados y probados en proyectos reales para garantizar que sean realmente útiles durante la formación académica y el desarrollo de prototipos.
